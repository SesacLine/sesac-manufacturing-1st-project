**실제 답변(API):**

```text
현재 고장 위험 진단 결과는 다음과 같습니다.

### 진단 요약
- **진단 상태**: 완료
- **현재 위험 수준**: 높음
- **신뢰도**: 높음

### 위험 분석
1. **OSF(과부하 계열)**: 높음
   - 계산: 공구 마모 215분 × 토크 62N·m = 13,330 ≥ M타입 한계 12,000 → 초과 (비율 1.11)
   - 영향 변수: 공구 마모, 토크, 제품 타입

2. **TWF(공구 마모 계열)**: 높음
   - 계산: 공구 마모 215분 → 200분 이상 (높음)
   - 영향 변수: 공구 마모

3. **HDF(열/냉각 계열)**: 중간
   - 계산: 온도차 11.0K (정상: 8.6K 이상) 및 회전속도 1320rpm (위험: 1380rpm 미만)
   - 영향 변수: 공기온도, 공정온도, 회전속도

4. **PWF(전원/구동 계열)**: 낮음
   - 영향 변수: 회전속도, 토크

### 조치 권장 사항
- OSF와 TWF가 높으므로, 공구 마모와 토크 조합을 확인하고, 공구 및 소재별 허용 부하를 점검해야 합니다.
- HDF는 중간 수준이므로, 공기온도와 공정온도를 모니터링하는 것이 좋습니다.
- 정지, 재가동, 정비 여부는 현장 안전 책임자와 설비 담당자가 최종 판단해야 합니다.

위험 요소를 충분히 고려하여 안전한 운영을 유지하시기 바랍니다.
```
